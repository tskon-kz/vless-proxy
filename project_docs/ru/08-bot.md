# Telegram-бот (`bot/bot.py`, `bot/strings.py`)

[English](../en/08-bot.md)

Реализован на aiogram 3. Доступ только для пользователей из `TG_ALLOWED_USER_IDS`.

## Команды

| Команда | Описание |
|---|---|
| `/status` | Статистика пула (active / dead / pending) и список запущенных прокси с портами и задержкой |
| `/check` | Принудительная проверка всех активных и pending серверов |
| `/help` | Справка с доступными командами |

Меню команд регистрируется в Telegram автоматически при старте через `bot.set_my_commands()`.

## Уведомления

Если задан `TG_NOTIFY_CHAT_ID`, бот отправляет сообщение в этот чат при смене статуса прокси **в процессе работы**:

- Прокси снова онлайн → `✅ Прокси онлайн: <имя> (<host>:<port>)\nЗадержка: <N>ms`
- Прокси упал → `💀 Прокси мертв: <имя> (<host>:<port>)\nОшибок подряд: <N>`

При старте, когда прокси проверяются впервые, уведомления **не отправляются** (нет предыдущего статуса — нет уведомления).

## `TG_BOT_PROXY`

Если Telegram заблокирован на сервере, задайте `TG_BOT_PROXY` для маршрутизации запросов к Telegram API через рабочий SOCKS5-прокси:

```env
TG_BOT_PROXY=socks5://127.0.0.1:10800
```
